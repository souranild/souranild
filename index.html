Project Directory - Souranil Das

Status: Released | Security: Internal

1. AI-Enabled Hydroponics Cultivation System

An automated IoT vertical farming design optimized for cultivation of medicinal crop yields. Configured hardware sensory nodes reporting EC, pH, water temperatures, and air metrics directly to Azure IoT Hub.

Data feeds were processed in Databricks where regression models adjusted lighting schedules and nutrient dosings automatically based on crop phase.

Tags: Azure IoT Hub, Databricks, Predictive AI, Hardware-Software Co-Design

2. SMADE - Smart Medical Assist Device

An elderly-healthcare assistive IoT capsule. Automates medication dispensing cycles, sound reminders, and channels real-time patient vitals telemetry streams to a web-based portal.

Tags: IoT Telemetry, Embedded Systems, Microcontrollers

3. async-cse Python Library

An asynchronous Python wrapper library interface querying the Google Custom Search JSON API. Engineered on top of asyncio to handle queries and return formatted JSON arrays.

Tags: Python, asyncio, API Wrapper, Open Source
