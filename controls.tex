\centerline{Play retro console games directly in your browser}

\section{GBA, SNES, and NES Controls}
To play games inside the Emulator window, make sure to click inside the game screen first to focus keyboard inputs. Use the following mapped keys:

\begin{itemize}
    \item \textbf{D-Pad (Movement)}: Arrow Keys OR W, A, S, D
    \item \textbf{Button A (Accept/Spin)}: X OR K
    \item \textbf{Button B (Back/Run)}: Z OR J
    \item \textbf{Left Shoulder (L)}: Q OR U
    \item \textbf{Right Shoulder (R)}: E OR I
    \item \textbf{Select Button}: Shift OR Space
    \item \textbf{Start Button}: Enter
\end{itemize}

\section{Nintendo DS Controls}
Nintendo DS games utilize the same keyboard controls as above, alongside Mouse clicks for touch-screen stylus inputs.

\section{Pro Tips}
\begin{itemize}
    \item If inputs do not register, click the black game screen area to refocus the controller module.
    \item Press Esc or C inside the Calculator app to clear its focus, which prevents general keys from being intercepted.
\end{itemize}
